# DEFINITION OF DONE (DoD) — ECOSSISTEMA DIGITAL

> Governança permanente, compartilhada com o Japão Relativo (ver `BRANDING/DEFINITION_OF_DONE.md` naquele repositório). Nenhuma tarefa é considerada concluída apenas porque "funciona" — só quando atende integralmente aos critérios abaixo.

## Integridade

- Zero imagens quebradas.
- Zero vídeos quebrados.
- Zero áudios quebrados.
- Zero PDFs quebrados.
- Zero downloads quebrados.
- Zero favicon quebrado.
- Zero manifest quebrado.
- Zero Open Graph quebrado.
- Zero caminhos inválidos para assets.
- Zero referências locais inexistentes.

## Conteúdo

- Todo artigo possui imagem válida.
- Todo card possui imagem válida.
- Todo hub possui imagem válida.
- Toda página possui SEO mínimo obrigatório.
- Todo conteúdo respeita o Brand Book e o Content System.

## UX

- Mobile validado.
- Desktop validado.
- Navegação consistente.
- Nenhum fluxo interrompido.
- Nenhum console error.

## Performance

- Assets otimizados.
- Lazy loading quando aplicável.
- Sem duplicação desnecessária.
- Sem arquivos órfãos criados pela tarefa.

## Arquitetura

- Componentes reutilizados sempre que possível.
- Nenhum padrão novo criado sem autorização.
- Estrutura de pastas preservada.
- Asset Registry atualizado quando necessário.

## Branding

- Tokens oficiais preservados (ver `DESIGN_SYSTEM.md`).
- Tipografia oficial preservada (Playfair Display + Lato + Cinzel).
- Paleta oficial preservada (dourado `#C9A84C` sobre azul-profundo).
- Linguagem editorial preservada (ver `EDITORIAL_GUIDELINES.md`).
- UX preservada (ver `UX_SYSTEM.md`).

## Critério adicional deste projeto

- Nenhuma entrega é considerada "publicada" só por existir no repositório local — produção (`procederfilosofico.com.br`) precisa refletir a mudança, com deploy explicitamente aprovado, para o critério de integridade valer de fato para o leitor real.

## Governança de status

Caso qualquer critério acima falhe, a tarefa recebe status:

**PENDENTE DE SANEAMENTO**

Somente quando todos os critérios forem atendidos o status pode mudar para:

**CONCLUÍDA**
