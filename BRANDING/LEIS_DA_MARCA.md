# LEIS DA MARCA — PROCEDER FILOSÓFICO

> 20 regras obrigatórias para qualquer artigo, página, hub, seção ou funcionalidade futura.

1. Toda página deve deixar pelo menos uma ideia que sobreviva à leitura — não apenas informação.
2. Nenhum título pode prometer lista de dicas; deve formular pergunta ou afirmação que provoque.
3. O dourado é declaração, não decoração — usar com escassez deliberada.
4. Nenhuma seção pode depender de urgência (contador, "última chance", "não perca").
5. Toda citação precisa de autoria e contexto — nunca solta, nunca anônima.
6. Humor é permitido como ironia fina; nunca como piada fácil ou meme.
7. Nenhum conteúdo finge neutralidade absoluta — o Proceder tem voz e ponto de vista.
8. Toda página deve poder ser lida em silêncio, sem som, sem autoplay ruidoso.
9. Nenhuma funcionalidade gamifica a leitura — sem streak, sem badge, sem pontuação.
10. O leitor nunca deve se sentir burro — todo conceito difícil precisa de uma porta de entrada simples.
11. Nenhum texto soa como conselho de produtividade ("use filosofia para ser mais eficiente").
12. Toda imagem de pensador transmite dignidade — nunca caricatura, meme ou estilo cartoon.
13. O site nunca parece estar vendendo algo de forma agressiva — CTA é convite, nunca interrupção.
14. Toda seção nova responde a: "isso ajuda alguém a pensar melhor, ou só a clicar mais?" — se for a segunda, descarta.
15. Proibido superlativo vazio ("o MELHOR ensaio sobre...", "você PRECISA ler isso").
16. Todo artigo devolve a pergunta para a vida do leitor no fechamento — nunca termina apenas na erudição.
17. Precisão histórica é inegociável, mesmo em conteúdo leve — erro factual sobre um pensador é falha de marca, não detalhe.
18. Nenhuma página depende de comparação social (ranking de leitura, "outros leitores também...").
19. Espaço vazio é parte do design tanto quanto qualquer elemento — nunca preencher uma página "porque sobrou espaço".
20. Toda novidade é testada com a pergunta: "isso pareceria fora de lugar numa biblioteca à meia-noite?" Se sim, não existe.
